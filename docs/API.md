# API Documentation

## Utils Module

### Functions

#### add(a, b)
Returns the sum of two numbers.

#### subtract(a, b)
Returns the difference of two numbers.

#### multiply(a, b)
Returns the product of two numbers.

## Helpers Module

### formatDate(date)
Formats a date to ISO format (YYYY-MM-DD).

### capitalize(str)
Capitalizes the first letter of a string.
